# Verified Recreation Map for Rhode Island Families

A self-initiated side project that transforms scattered parent and community recommendations into verified geospatial datasets and interactive maps.

## Project Overview

I noticed that simple family outing questions were surprisingly hard to answer:

* Where are nearby splash pads?
* Which outdoor gyms actually exist and are usable?
* Which places are worth a short drive?
* Can I plan visually instead of rereading old recommendation threads?

Information was fragmented across Facebook groups, community discussions, outdated posts, and inconsistent public sources.

To solve that, I built a structured location dataset and interactive map experience for Rhode Island recreation planning.

---

## What I Built

This project includes:

* Verified **GeoJSON datasets**
* Interactive maps for:

  * Outdoor Gyms
  * Splash Pads
* Standardized location records
* Manual validation of names, addresses, and coordinates
* Reusable structure for future expansion

---

## Data Collection Process

Locations were gathered from publicly shared community recommendations, then verified through:

* Google Maps
* Public websites
* Local references
* Duplicate cleanup
* Coordinate confirmation

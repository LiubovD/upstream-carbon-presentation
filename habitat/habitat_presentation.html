Mapping Favorable Chanterelle Habitat in Rhode Island

A side project using public soils data, habitat classes, and GIS overlays to identify where favorable mushroom habitat conditions exist before going into the field.

View GeoJSON on GitHub

Problem & Motivation

Chanterelles do not grow randomly. They depend on forest type, host trees, drainage, soil texture, and landscape position. Most foraging relies on trial and error, so I wanted to test whether public GIS data could narrow the search.

Loading habitat map...

This web map displays a lightweight sample of the full favorable habitat layer so it renders quickly in the browser. The full GeoJSON is stored in the repository.

Approach

  • Selected favorable upland soils such as ChC, HkC, CaD, LgC, SuB, and SvB.
  • Selected forest habitats such as Oak Forest, Hemlock/Hardwood Forest, and Mixed Forest.
  • Intersected soil and habitat polygons in GIS.
  • Exported the resulting favorable habitat layer as GeoJSON.
  • For web display, sampled the layer to keep browser rendering fast.

Action Taken

I pushed the resulting GeoJSON to GitHub and turned it into an interactive map. The output is a practical scouting layer: a way to focus field time on places where soil and forest conditions align.
